# CoraFlow

一个基于 Vue 3 的可视化工作流编辑器，需搭配python自动化脚本工具使用，实现自动化脚本工作流。

### 演示：[https://coraflow.ai0859.cn/](https://coraflow.ai0859.cn/)

## 功能特性

- 可视化拖拽节点编辑器
- AI对话创建工作流
- 支持多种自动化操作节点
- 节点连线与条件分支
- 工作流导入/导出 (JSON 格式)

## 安装

```sh
npm install
```

## 开发

启动开发服务器：

```sh
npm run dev
```

## 使用

- Shift+鼠标左键：框选多个节点
- Ctrl+s：保存当前流
- 鼠标右键：插入Group框
